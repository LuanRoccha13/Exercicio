Um pouco sobre HTML

O HTML (HyperText Markup Language) é a linguagem de marcação fundamental da web. Ele utiliza uma estrutura de tags para definir elementos como títulos, parágrafos e links. Sua função principal é organizar o conteúdo e a hierarquia de uma página online. Não é uma linguagem de programação, mas sim uma base estrutural para documentos. Através dele, navegadores conseguem interpretar e exibir informações corretamente aos usuários. O padrão atual é o HTML5, que suporta recursos nativos de áudio, vídeo e gráficos. Para estilização e interatividade, ele trabalha em conjunto com tecnologias como CSS e JavaScript.


Alguns conceitos aprendidos durante o módulo

h1

a
h6
: Definem títulos e subtítulos, sendo o h1 o mais importante e o h6 o de menor hierarquia.

p: Cria um parágrafo de texto comum.

mark: Destaca um trecho do texto com um fundo colorido (efeito marca-texto).

small: Deixa o texto com uma fonte menor, usado para observações ou direitos autorais.

i: Aplica estilo itálico ao texto.

u: Adiciona um sublinhado ao texto selecionado.

strong: Deixa o texto em negrito, indicando importância semântica.

ol, ul e li: ol cria listas numeradas, ul listas com marcadores e li define cada item da lista.

a: Cria links para outras páginas ou arquivos usando o atributo href.

hr: Desenha uma linha horizontal para separar seções do conteúdo. (conforme mostrado abaixo)

sub: Define texto subscrito (abaixo da linha, como em H₂O).

sup: Define texto sobrescrito (acima da linha, como em 2²).

blockquote: Indica uma citação longa retirada de outra fonte.

Exemplo de citação

font: Tag antiga (obsoleta no HTML5) usada para alterar cor, tamanho ou face da fonte; hoje prefere-se CSS.

del: Risca o texto para indicar que foi removido ou excluído.

abbr: Define uma abreviação; ao passar o mouse, exibe o significado completo (usando o atributo title).
